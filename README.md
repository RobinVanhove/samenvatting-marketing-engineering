# How to compile
Normaalgezien bevat [de pdf](samenvatting.pdf) de laatst gecompileerde versie.

Om nieuwe aanpssingen te compileren gebruik [pandoc](http://pandoc.org/)

```sh
$ pandoc samenvatting.md -o samenvatting.pdf
```
